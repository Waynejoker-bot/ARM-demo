# Historical Documentation Archive

这里存放已经归档的历史文档。

## 归档规则

历史文档被保留，是为了回答：

- 当时为什么会这样想
- 为什么后面又发生转向
- 哪些方案已经被吸收，哪些已经被放弃

## 如何阅读历史文档

所有归档文档都应被视为：

- 历史上下文
- 阶段性设计快照
- 复盘材料

它们不再是当前 live spec。

如果你要开始新的实现，请先回到：

- `AGENTS.md`
- `docs/current/README.md`
- `docs/current/product-direction.md`
- `docs/current/implementation-status.md`
